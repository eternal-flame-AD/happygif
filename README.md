一个gif加字幕的工具

## Usage

- $python happygif.py
- show [frameid] 显示对应帧
- drawframes 绘制文字
- write 输出文件
- setfont 设置字体
- setpos 设置位置
- setanchor 设置对齐
- setcolor 设置颜色
